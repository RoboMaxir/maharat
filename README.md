# سیستم هوش مصنوعی داخلی (On-Device AI) برای اپلیکیشن فریلنسری

## توضیحات پروژه

این پروژه یک سیستم هوش مصنوعی کاملاً آفلاین برای یک اپلیکیشن فریلنسری ایرانی است که بدون نیاز به اینترنت یا APIهای خارجی کار می‌کند. سیستم به گونه‌ای طراحی شده که کاملاً با محدودیت‌های ایران سازگار باشد و از زبان فارسی به خوبی پشتیبانی کند.

## ویژگی‌های اصلی

- **کارکرد کاملاً آفلاین**: تمام عملیات‌ها روی دستگاه کاربر انجام می‌شود
- **پشتیبانی کامل از فارسی**: با توجه به محدودیت‌های NLP در فارسی
- **حجم کم کد**: کمتر از 500 خط کد
- **سازگار با Flutter 3.10+ و Dart 3.0+**
- **عدم ذخیره داده‌های شخصی**: همه چیز در حافظه محلی ذخیره می‌شود

## چهار موتور اصلی

### 1. مذاکره هوشمند (Negotiation Engine)
- محاسبه قیمت پیشنهادی بر اساس بازار ایران
- در نظر گرفتن تجربه کاربر و پیچیدگی پروژه
- پیشنهاد استراتژی مذاکره مناسب
- پایگاه دانش محلی با نرخ‌های رایج در ایران

### 2. تشخیص کلاهبرداری (Fraud Detection Engine)
- شناسایی الگوهای مشکوک در پروژه‌ها و کارفرمایان
- بدون نیاز به داده‌های ابری
- استفاده از الگوهای شناخته‌شده کلاهبرداری در بازار ایران
- ارائه نکات ایمنی و توصیه‌های عملی

### 3. یادگیری تطبیقی (Adaptive Learning Engine)
- تولید چالش‌های آموزشی بر اساس رفتار کاربر
- پایگاه دانش محلی با مثال‌های ایرانی
- تنظیم سطح دشواری بر اساس تجربه کاربر
- پیگیری پیشرفت و ارائه بازخورد

### 4. مربی هوشمند (Smart Mentor Engine)
- پاسخ به سوالات رایج فریلنسری با پایگاه دانش محلی
- جستجوی هوشمند در بانک اطلاعاتی سوالات
- تنظیم پاسخ بر اساس سطح تجربه کاربر
- ارائه پیشنهادات آموزشی مرتبط

## ساختار پروژه

```
lib/
├── engines/
│   ├── negotiation_engine.dart      # موتور مذاکره هوشمند
│   ├── fraud_detection_engine.dart  # موتور تشخیص کلاهبرداری
│   ├── adaptive_learning_engine.dart # موتور یادگیری تطبیقی
│   └── smart_mentor_engine.dart     # موتور مربی هوشمند
└── on_device_ai_system.dart         # سیستم کلی که همه موتورها را ادغام می‌کند
```

## نحوه استفاده

برای استفاده از سیستم، کافی است فایل `on_device_ai_system.dart` را در پروژه Flutter خود وارد کنید:

```dart
import 'lib/on_device_ai_system.dart';

void main() {
  final aiSystem = OnDeviceAISystem();
  
  var result = aiSystem.processFreelanceProject(
    projectDetails: {
      'id': 'proj_123',
      'type': 'web_development',
      'complexity': 3,
      'deadline': 14,
      'description': 'سایت فروشگاهی فوری - کار تست لازم است - پرداخت بعد از تحویل',
      'paymentTerms': 'پس‌پرداخت',
      'currentOffer': 1200000.0,
      'hasQuestions': true,
      'question': 'چطور قیمت پروژه خود را تعیین کنم؟',
    },
    clientDetails: {
      'history': 'new',
      'type': 'local',
      'contactMethod': 'telegram',
    },
    userProfile: {
      'experienceLevel': 3,
      'specialty': 'web_development',
      'needsTraining': true,
      'recentPerformance': 0.7,
    },
  );
  
  print('قیمت پیشنهادی هوشمند: ${result['smartPricing']['recommendedPrice']} تومان');
  print('سطح ریسک کلی: ${result['overallRisk']}');
}
```

## محدودیت‌ها و راه‌حل‌ها

1. **محدودیت NLP فارسی**: به جای مدل‌های پیچیده NLP، از الگوریتم‌های ساده تطبیق الگو و کلمات کلیدی استفاده شده است
2. **محدودیت حجم مدل**: استفاده از کتابخانه‌های سبک و الگوریتم‌های ساده به جای مدل‌های عمیق
3. **عدم دسترسی به APIهای خارجی**: تمام داده‌ها در پایگاه دانش محلی ذخیره شده‌اند
4. **محدودیت حافظه دستگاه**: کد بهینه‌سازی شده برای حداقل مصرف منابع

## مثال‌های کاربردی

سیستم شامل نمونه‌های کاربردی برای هر موتور است که نشان می‌دهد چگونه می‌توان از هر موتور در صفحات مختلف اپلیکیشن استفاده کرد:

- **صفحه پیشنهاد قیمت**: استفاده از موتور مذاکره هوشمند
- **صفحه بررسی پروژه**: استفاده از موتور تشخیص کلاهبرداری
- **بخش آموزش**: استفاده از موتور یادگیری تطبیقی
- **بخش سوالات متداول**: استفاده از موتور مربی هوشمند

## نکات مهم

- تمام متن‌ها به زبان فارسی و با توجه به فرهنگ کاری ایران نوشته شده‌اند
- سیستم بدون نیاز به اینترنت قابل استفاده است
- حداکثر تلاش شده است که از کتابخانه‌های خارجی و سنگین استفاده نشود
- تمام داده‌ها به صورت محلی و رمزنگاری‌شده ذخیره می‌شوند
